    # 自我介绍

    # 2019

    1. 基本情况：您好，我的名字叫涂阿燃，毕业于华南师范大学，专业是多媒体与网络技术（简单说就是将技术应用在教育上，技术指的是网络技术和多媒体技术。）

    毕业后，我一直从事前端开发这个岗位。主要的技术栈是很vue和小程序,熟悉h5的移动端兼容、bootstrap框架、也熟悉flex布局等常用的css3属性；在vue方面，熟悉vue-cli项目搭建、webpack打包相关，熟悉vuex组件通信、vue-router路由跳转，用过饿了吗框架，了解mpvue，然后从今年三月份开始，主攻uniapp框架（类似于taro），是多端统一的开发的解决方案。

    1. 项目介绍：经手过的项目有：数据统计系统、B2B商城模板开发、O2O医护到家小程序、校园二手交易小程序。

    想法观点：平常爱逛技术社区，喜欢写写文章。我认为：技术是手段、需求是目的，格局决定上限，学习决定下限。技术是为产品来服务的，开发前的总体设计一定要清晰，然后就是要保持不断的学习。对于以后的规划就是，学好技术，然后拥有一个更好的视野，能够负责有价值的项目。

    项目：

    • 静态web页面，H5移动端兼容
    • 数据统计系统
    • B2B商城模板开发
    • O2O医护到家小程序
    • 二手商城交易小程序

    # 2020

    1. 您好，我的名字叫涂阿燃，于2018年毕业于华南师范大学，专业是多媒体与网络技术。

    2. 毕业后，一直从事前端开发这个岗位，目前在多益网络任职。主要的技术栈是 vue 和小程序,熟悉 vue2 全家桶和 element ui、echart 框架。

    3. 参与过文件日志系统、后台管理系统的开发，也做过 B2B 商城和小程序，也做过物联网 h5 项目。

    项目：

    • 日志系统
    • 安全展示平台
    • 梯控系统
    • B2B商城
    • uniapp小程序
